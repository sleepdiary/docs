# Software

Here are some programs to help you track your sleep. The galleries below all represent the same data, taken from our common sleep diariesopen in new window.

# Fitbit

Fitbit

A range of devices for tracking fitness and other statistics

Home page
Device with free apps

# Procedure

  1. buy a Fitbit device
  2. download the Android appopen in new window or iOS appopen in new window
  3. link your device with the app, so the device can upload reports through it
  4. use the app or the website's sleep pageopen in new window
